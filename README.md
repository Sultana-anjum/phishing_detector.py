# Phishing Email Detection Model

This project uses Machine Learning to classify emails as phishing or safe.

## Features
- Detects phishing emails
- Uses TF-IDF + Logistic Regression
- Shows accuracy and confusion matrix

## Technologies
- Python
- Scikit-learn
- Pandas

## How to Run
1. Install dependencies
2. Run: python phishing_detector.py
3. Enter email text to test

## Output
- Predicts whether email is Safe or Phishing
