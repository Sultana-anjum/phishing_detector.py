# Phishing Email Detection Model

This project uses Machine Learning to classify emails as phishing or safe.

## Features
- Text-based email classification
- Uses TF-IDF for feature extraction
- Logistic Regression model
- Displays accuracy and confusion matrix

## Technologies Used
- Python
- Scikit-learn
- Pandas

## How to Use
1. Run the script
2. Enter email text
3. Get prediction (Phishing or Safe)

## Purpose
To understand machine learning applications in cybersecurity.
